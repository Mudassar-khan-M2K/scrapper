# 🇵🇰 Pakistan Jobs Bot v4.0

> Developed by **Mudassar Khan** | 📲 +92 347 7262704

Ek automated WhatsApp Bot jo Pakistan ke sarkari aur private jobs har 20 minute mein automatically dhundh kar aap tak pahunchata hai!

---

## ✨ Features

- 🤖 **13 Job Sources** — NJP, Punjab, FPSC, NADRA, PAEC, Army, Navy, PAF, ANF, Rozee, Engro, PSO, Mustakbil
- 📢 **1 Job = 1 Message** — Channel pe har job alag message mein
- 🎓 **Education Filter** — Matric / Inter / BS / MS
- ⚔️ **Defence Jobs** — Army, Navy, PAF, ANF
- 🏛️ **Govt Jobs** — Federal + Punjab
- 🏢 **Private Jobs** — Top Pakistani companies
- 📊 **Live Dashboard** — Heroku URL pe real-time stats
- 🔕 **No Spam** — Sirf valid commands ka reply
- 24/7 Auto reconnect

---

## 🚀 Deploy on Heroku

```bash
git init
git add .
git commit -m "Pakistan Jobs Bot v4.0"
heroku create your-bot-name
git push heroku main
heroku config:set SESSION_ID="Gifted~YOUR_SESSION_HERE"
heroku config:set NODE_ENV=production
heroku restart
heroku logs --tail
```
